## Design Context

### Users
AI/ML 方向的技术面试官和招聘经理。他们在快速浏览候选人的项目时，需要一眼看出"这人能做产品级别的东西"。用户同时包括陈昊楠本人——需要在面试中演示 RAG 问答流程。界面要让人觉得：这是一个真正在用的产品，不是课程作业。

### Brand Personality
温暖、专业、克制。三个词：**Warm · Precise · Grounded**

情绪目标：访问者应该感到"这个人对细节有品味"——就像走进一家灯光恰到好处的书店，而不是一个 LED 闪烁的电竞网吧。

### Aesthetic Direction
- **参考**: Claude/Anthropic 官网 — 暖奶油色底 (#F5F0E8) + 赤陶橙 (#DA7756) 双色系
- **反面参考**: 冷灰科技感 dashboard、深色 hacker terminal 风格、任何纯黑+霓虹配色
- **色彩规则**: 拒绝灰色。所有中性色必须带暖色调（黄/橙底色）。只用两个色系：暖奶油 + 赤陶橙
- **模式**: 亮色优先（light mode 为默认），暗色模式保留但也必须温暖
- **排版**: 干净、大量留白、Geist 字体系

### Design Principles

1. **暖而不灰** — 每一个"灰色"都必须带暖色调。背景是奶油色不是白色，文字是暖棕不是纯黑，边框是淡褐不是银灰。
2. **双色克制** — 只有奶油色（底色系）和赤陶橙（强调色系）两个色系。不引入蓝色、紫色、绿色。少即是多。
3. **内容优先** — 设计服务于 RAG 问答体验和论文/博客内容展示，不做装饰性动效。
4. **中文为主** — 界面文案默认中文，代码和技术术语保持英文。
5. **亮色模式** — 默认使用亮色主题，像 Claude 官网一样，明亮温暖而非暗沉冷峻。
